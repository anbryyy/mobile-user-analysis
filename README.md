# Mobile User Behavior Analysis

## Project Overview
Analysis of mobile app user behavior to identify engagement patterns, segment users, and generate product insights.

## Dataset
Mobile usage data including:
- App usage time
- Screen time
- Data usage
- Battery consumption
- Device and OS info
- Demographics

## Tools
- Python
- Pandas
- Matplotlib / Seaborn
- Google Colab

## Analysis
- User segmentation (low, medium, high activity)
- Power users identification (top 10%)
- Behavior comparison across segments
- Correlation analysis of key metrics
- Device and demographic analysis

## Key Insights
- Most users are low-activity users
- Small group of power users drives majority of engagement
- Higher usage correlates with higher battery and data consumption
- Clear behavioral differences between segments

## Dashboard
Built in Python with visualizations:
- Segment distribution
- Average usage metrics
- Resource consumption comparison
- Correlation heatmap

## Conclusion
The analysis provides actionable insights for improving user retention, segment-based product strategy, and monetization of high-engagement users.
